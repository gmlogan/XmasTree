# Christmas Tree Images

Place your Christmas tree PNG images in this directory with the following names:

## Required Images

1. **xmas_tree_indoor_on.png** - Indoor tree when lights are ON (colorful/lit)
2. **xmas_tree_indoor_off.png** - Indoor tree when lights are OFF (darker/grayscale)
3. **xmas_tree_outdoor_on.png** - Outdoor tree when lights are ON (colorful/lit)
4. **xmas_tree_outdoor_off.png** - Outdoor tree when lights are OFF (darker/grayscale)

## Image Specifications

- **Format**: PNG (preferred) or SVG
- **Recommended size**: 90x120 pixels (or higher resolution - will be auto-resized)
- **Color mode**: RGB565 (will be converted automatically)
- **Style**: Consider using bright, colorful versions for "on" state and grayscale/dimmed versions for "off" state

## Tips

- You can use the same tree image for both indoor and outdoor if preferred
- For "off" state, you can desaturate the "on" image or use a darker version
- Transparent backgrounds work well with the dark page background (0x111827)
